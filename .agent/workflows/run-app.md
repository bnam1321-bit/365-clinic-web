description: 가 개발된 홈페이지를 로컬에서 실행하는 방법입니다.
---

// turbo
1. 터미널에서 프로젝트 디렉토리로 이동합니다 (이미 `c:\note\premium-clinic-web`인 경우 생략 가능).

2. 개발 모드로 실행하려면 다음 명령어를 입력합니다:
   ```powershell
   npm run dev
   ```

3. 브라우저에서 `http://localhost:3000`에 접속하여 확인합니다.

4. 프로덕션 빌드 후 실행하려면 다음 명령어를 입력합니다:
   ```powershell
   npm run build
   npm run start
   ```
